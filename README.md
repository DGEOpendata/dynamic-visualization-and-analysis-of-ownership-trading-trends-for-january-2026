# Dynamic Visualization and Analysis of Ownership Trading Trends for January 2026

## Overview
This project provides an interactive web application to analyze and visualize the `Ownership Trading Summary for January 2026` dataset. The application is designed to enhance accessibility and engagement by offering dynamic charts, question-based insights, and downloadable reports.

## Features
- **Dynamic Visualizations**: Interactive charts generated using Plotly.
- **Real-time Data Fetching**: Automatically fetches the latest dataset from the Abu Dhabi Open Data Platform.
- **User-friendly Interface**: Intuitive navigation for non-technical users.
- **Downloadable Insights**: Export insights in Excel and PDF formats.

## Prerequisites
- Python 3.8+
- Flask (`pip install flask`)
- Pandas (`pip install pandas`)
- Plotly (`pip install plotly`)

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/ownership-trading-visualization.git
   
2. Navigate to the project folder:
   bash
   cd ownership-trading-visualization
   
3. Install required Python libraries:
   bash
   pip install -r requirements.txt
   

## Usage
1. Run the Flask app:
   bash
   python app.py
   
2. Open a browser and navigate to `http://127.0.0.1:5000`.
3. Explore the dataset through the interactive charts and download insights as needed.

## Deployment
To deploy the application, consider using platforms like Heroku or AWS Elastic Beanstalk. Follow their respective deployment guides for Flask applications.
